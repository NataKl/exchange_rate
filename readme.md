Что добавлено:
Импорт colorama (строка 6) — для цветного вывода
1. Функция get_available_base_currencies() (строки 109-119):
Возвращает список базовых валют из файла
2. Функция find_base_currency_for_currency() (строки 122-140):
Находит базовую валюту, в rates которой есть указанная валюта
3. Функция get_rates_for_base_currency() (строки 143-180):
Если валюта является базовой в файле — возвращает её rates напрямую
Если нет — вычисляет курсы через другую базовую валюту
4. Функция display_currency_rates() (строки 183-210):
Отображает курсы валют в табличном формате с цветами
Выводит по 4 валюты в строке
5. Функция show_currency_rates_interface() (строки 213-250):
Интерактивный интерфейс для ввода базовой валюты
Показывает доступные базовые валюты
Позволяет вводить любую валюту (не только базовую)
К
ак работает:
Загружает данные из файла (или обновляет, если файл старше 24 часов)
Показывает доступные базовые валюты
Запрашивает ввод базовой валюты
Вычисляет курсы всех валют относительно выбранной базовой
Отображает курсы в табличном формате

Пример использования:

Введите код базовой валюты: USD
→ Покажет все курсы относительно USD

Введите код базовой валюты: KZT
→ Вычислит курсы через USD (если KZT есть в rates USD) и покажет все валюты относительно KZT
